# FireTask
En este archivo .rar encontrarás todos los archivos del proyecto FireTask que consiste en mediante las librerias de JFrame, JPanel y Canvas, generar una animación de un fuego.

La ventana utiliza un GridbagLayout en el que aparecen 3 botones. Uno para pintar el fondo, otro para pintar la animación del fuego y si se pulsa otra vez se detiene, y el otro para borrar el fuego. 
Y el canvas donde se pintan las imagenes.

El fuego se pinta pintando una imagen pixel a pixel utilizando una paleta de colores y una matriz de temperaturas, según la temperatura en ese pixel elige un color representado en la paleta de colroes
y lo pinta en su posición correspondiente en la imagen.
